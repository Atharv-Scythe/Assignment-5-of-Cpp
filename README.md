# C++ Programming Assignment – 5

## 📌 Description
This repository contains solutions to **Assignment 5 of C++ Programming**.  
The assignment focuses on **arrays, functions, parameter passing techniques, pointers, and advanced C++ concepts**.

All programs are written in **C++ (.cpp)** and compiled using **g++ (GCC compiler)**.

---

# 🛠 Concepts Used
- Arrays (`[]`, pointer notation, base address notation)
- Functions
- Parameter passing:
  - By Value
  - By Reference
  - By Address
- Pointers
- References
- Loops (`for`, `while`)
- Conditional statements
- Mathematical operations
- Static variables
- Enumeration
- I/O formatting (`setw`, `endl`)
